# flyer-springmvc-rest是什么？
flyer-springmvc-rest是一个Java样例项目，这里维护了flyer-maker的模板。通常来说，这里的代码更新会在1周内同步至flyer-maker。
了解更多请访问[flyer-maker](https://github.com/vancefantasy/flyer-maker)
